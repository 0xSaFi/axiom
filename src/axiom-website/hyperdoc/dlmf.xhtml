The Gamma Function by R. A. Askey and R. Roy

The Gamma function is an extension of the factorial function to real and complex numbers. For positive integers, Γ ( n ) = ( n - 1 ) ! .

These pages explore Axiom's facilities for handling the Gamma function. In particular we try to show that Axiom conforms to published standards.
