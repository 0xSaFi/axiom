Gamma Function

Γ ( n + 1 ) = n !

Psi Function

ψ ( 1 ) = - γ
ψ ( 1 2 ) = - γ - 2 ln 2
ψ ( n + 1 ) = ∑ k = 1 n 1 k - γ
ψ ( n + 1 2 ) = - γ - 2 ln 2 + 2 ( 1 + 1 3 + … + 1 2 n - 1 ) n ≥ 1
ℑ ψ ( ⅈ y ) = 1 2 y + π 2 coth ( π y )
ℑ ψ ( 1 2 + ⅈ y ) = π 2 tanh ( π y )
ℑ ψ ( 1 + ⅈ y ) = - 1 2 y + π 2 coth ( π y )

0 < p < q are integers, then

ψ ( p q ) = - γ - ln q - π 2 cot ( π p q ) + 1 2 ∑ k = 1 q - 1 cos ( 2 π k p q ) ln ( 2 - 2 cos ( 2 π k q ) )
