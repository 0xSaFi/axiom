Bohr-Mollerup Theorem


If a positive function f ( x ) on ( 0 , ∞ ) satisfies f ( x + 1 ) = x f ( x ) , f ( 1 ) = 1 , and ln f ( x ) is convex, then f ( x ) = Γ ( x ) .
