The main functions treated in this chapter are the gamma function Γ ( z ) ,the psi function ψ ( z ) ,the beta function B ( a , b ) , and the q -gamma function Γ q ( z ) .

The notation Γ ( z ) is due to Legendre. Alternative notations for this function are: Π ( z - 1 ) (Gauss) and ( z - 1 ) ! . Alternative notations for the psi function are:
